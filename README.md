# Daily Task Tracker

A minimalist, responsive web application to manage daily tasks. Built with HTML5, CSS3, and Vanilla JavaScript.

## 🚀 Features
- **Add Tasks:** Quick entry for daily goals.
- **Persistence:** Uses `localStorage` to save tasks even after refreshing the browser.
- **Constraints:** Maximum of 15 tasks to prevent clutter; prevents empty task entries.
- **Task Management:** Mark tasks as complete or delete them when finished.
- **Filtering:** Filter tasks by 'All', 'Pending', or 'Completed' status.
- **Edit Capability:** Modify existing tasks on the fly.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Storage:** Web Storage API (localStorage)
- **Deployment:** GitHub Pages
